# `vim/vi` Editor:

1. Save and Quit: `:wq`
2. Save: `:w`
2. Quit: `:q!`
3. Insert Text: `i`
4. Delete Line: `:d` (single) or, `eacape + dd` (multiple)
5. Undo Text  : `:u` (single) or, `eacape + uu` (multiple)


**Note-1:** Write `:` by using `escape + shift + :`. <br/>
**Note-2:** Write any command by using `escape` first.
