# `vim/vi` Editor:

1. Save and Quit: `:wq`
2. Save: `:w`
3. Quit: `:q!`
4. Insert Text: `i`
5. Delete Line: `:d` (single) or, `eacape + dd` (multiple)
6. Undo Text: `:u` (single) or, `eacape + uu` (multiple)

&nbsp;

**Note-1:** Write `:` by using `escape + shift + :`. <br/>
**Note-2:** Write any command by using `escape` first.
